🚀 COMO COMEÇAR

Passo 1 - Instalar
   $ cd FechamentoJobinho
   $ npm install

Passo 2 - Iniciar
   $ npm run dev
   
Passo 3 - Usar
   → Navegador abrirá em http://localhost:5173
   → Clique em "Começar"
   → Crie seu primeiro fechamento!
