💡 EXEMPLO DE USO
═══════════════════════════════════════════════════════════════════════════════

1. Clique em "Começar"

2. Preencha:
   Nome: "Jobinho"
   Período: "01/09 à 30/09"
   Mês: "Setembro"

3. Adicione dias (ex: 02/09, 03/09, 04/09...)

4. Para cada dia, clique "+ Adicionar Serviço"
   Preencha: Veículo, Placa, Valor, Descrição

5. Totais atualizam automaticamente ✨

6. Clique "📊 Visualizar Relatório"

7. Imprima ou exporte para PDF

  • fechamento_jobinho_closings  ← Todos os fechamentos

Estrutura:
  MonthlyClosing
  ├─ id
  ├─ title ("Jobinho")
  ├─ period ("01/09 à 30/09")
  ├─ month ("Setembro")

❓ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problema: "npm: command not found"
Solução: Instale Node.js em https://nodejs.org/

Problema: Porta 5173 em uso
Solução: npm run dev -- --port 3000

Problema: Dados não aparecem
Solução: F12 → Application → LocalStorage → ver chaves

Problema: Layout estranho
Solução: F12 → Toggle Device Toolbar (Ctrl+Shift+M)
